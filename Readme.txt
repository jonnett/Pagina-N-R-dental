Se agregó un formulario de contacto para que los pacientes puedan enviar mensajes directamente al correo y resolver dudas de manera más rápida y segura. También se implementó protección contra ataques XSS mediante validaciones y sanitización de datos en JavaScript, mejorando así la seguridad del sitio web.
Los especialistas fueron movidos desde el HTML hacia JavaScript utilizando arreglos de objetos y renderizado dinámico, permitiendo una estructura más limpia, organizada y difícil de modificar directamente desde el HTML. Además, se añadieron filtros interactivos en el buscador de especialistas, permitiendo filtrar por área dental y mejorar la experiencia del usuario. Gran parte del contenido ahora es generado dinámicamente desde JavaScript, lo que redujo considerablemente el código repetido en el HTML y facilitó la mantención del proyecto.
Se incorporaron etiquetas dinámicas de accesibilidad para mejorar la navegación de personas con discapacidad visual y optimizar la accesibilidad general del sitio.
Por último, se mejoró el sistema de modo oscuro y modo claro, agregando persistencia con localStorage para guardar automáticamente la preferencia visual del usuario.
